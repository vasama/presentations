Problems with static initalisers

  • Runtime overhead
  • Dynamic allocation
  • Fragile linking
  • Initialisation order
  • Impedes optimisation

Alternative solutions?

Linker sections 
							
								#include <test_framework.hpp>

								__attribute__((section("test_framework_test_case_list")))
